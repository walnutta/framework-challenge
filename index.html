How it works

1. Choose a protocol

Select a vetted cloning protocol from the library or import your own. Each protocol includes required reagents and steps.

2. Prepare & track

Assign samples, print labeled tubes, and log metadata. Every sample update is recorded for traceability. Physical hardware can be customized to fit your workflow.

3. Run & report

Execute the run (manual or automated), collect results, and generate a downloadable report with audit trails.

Safety & ethics

CloneLab is intended for legitimate scientific and educational use within authorized environments. Users must comply with institutional biosafety policies and applicable laws. Administrators can restrict features and enforce approvals.
